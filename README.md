# OuvintePro · Painel

Painel de ouvintes da Rádio Cidade FM 102,7. MVP com dados ilustrativos.

## Stack

- Next.js 14 (App Router)
- Tailwind CSS 3
- Recharts
- TypeScript

## Rodar localmente

```bash
npm install
npm run dev
```

Abra http://localhost:3000.

## Build

```bash
npm run build
```
